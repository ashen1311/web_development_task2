    Campus Ride Share

    HTML5CSS3JavaScript 2024

    A carpooling web app I built for a university module. Students can post or find rides between campus and the city. The brief said "make a form" — I turned it into a properly semantic, keyboard-navigable interface because it felt wrong not to.

    What I focused on

    • Full keyboard navigation — every action reachable without a mouse
    • Semantic form with grouped inputs and descriptive labels
    • ARIA live region announces newly posted ride listings
    • Lighthouse Accessibility score: 100
    • HTML5
    • CSS3
    • JavaScript
    • ARIA

    Task Board

    JavaScriptCSS Grid 2024

    A Kanban task manager. You can drag cards between columns or do everything with a keyboard too — which was the hard part. I implemented the ARIA drag-and-drop pattern from the W3C guide and learned more about focus management than anything else I've done.

    What I focused on

    • Keyboard drag-and-drop using ARIA grabbed / dropeffect
    • Status announcements via aria-live="assertive"
    • localStorage persistence — tasks survive a refresh
    • CSS Grid for column layout — no frameworks at all
    • JavaScript
    • CSS Grid
    • localStorage
    • ARIA
